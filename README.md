# `commands` a set of useful scripts

* conf 
```
conf <program_name>
```
<p>conf: takes the annoying step of cding into the .config directory taking you straight a program's config directory though nvim.</p>

* hex (binary)
```
hex <decimal_value>
```
<p>hex: converts any decimal integer to its hexadecimal representation.</p>

* initgit
```
initgit
```
<p>initgit: puts all the content in your current directory in a git repo of the same name, given it exists.</p>

* zipgbr
```
zipgbr
```
<p>zipgbr: takes all the created .gbr and .drl files from plotting a PCB zips them, moves the zip to the ./zips directory, and removes the .gbr and .drl files.</p>
