# `commands` a set of useful scripts

* conf 
`
conf <program_name>
`
*conf* takes the annoying step of cding into the .config directory and taking you straight a program's config directory though nvim.

* hex (binary)
`
hex <decimal_value>
`
*hex* converts any decimal integer to its hexadecimal representation. 

* initgit
`
initgit
`
*initgit* puts all the content in your current directory in a git repo of the same name, given it exists.

* zipgbr
`
zipgbr
`
*zipgbr* takes all the created .gbr and .drl files from plotting a PCB zips them, moves the zip to the ./zips directory, and removes the .gbr and .drl files.
